    <p>A basic proxy scraper.</p>
</div>

> ### Setup ✨
### Cloning the git respository
```
$ git clone https://github.com/bl1ngbl1ng/proxiD
$ cd proxiD
```
### Compiling
```
$ dmd proxi.d
```
### Running
```
$ ./proxi <proxy-type>
```
